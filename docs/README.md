## Overview

This repository contains the **documentation site** for the [Aura Stack Auth](https://github.com/aura-stack-ts/auth) ecosystem.  
It is built with **MDX** and served through **[FumaDocs](https://fumadocs.dev)** — a developer-focused documentation framework that enables interactive examples, type-safe code blocks, and consistent UI components.

## Getting Started

You can run the example app locally for development or contribution.

### From the repository root

```bash
pnpm web:docs
```

### Or manually from the docs directory

```bash
cd docs
pnpm install
pnpm dev
```

Once started, open your browser at `http://localhost:3000` to view the docs.
