## Overview

This project contains a demonstration of Aura Auth authentication with [OAuth 2.0 providers](https://aura-stack-auth.vercel.app/docs) with [React Router v7](https://reactrouter.com/) app

## Getting Started

You can run the documentation locally for development or contribution.

### From the repository root

```bash
pnpm dev --filter=react-router
```

### Or manually from the root directory

```bash
cd apps/react-router
pnpm install
pnpm dev
```

Once started, open your browser at `http://localhost:5173/` to view the app.
