## Overview

This project demonstrates Aura Auth authentication using [OAuth 2.0 providers](https://aura-stack-auth.vercel.app/docs/) in an [Tanstack Start](https://tanstack.com/start/latest) application.

## Getting Started

You can run the documentation site locally for development or contribution.

### From the repository root

```bash
pnpm dev --filter=tanstack-start
```

### Or manually from the root directory

```bash
cd apps/tanstack-start
pnpm install
pnpm dev
```

Once started, open your browser at `http://localhost:3000/` to view the app.
