## Overview

This project demonstrates Aura Auth authentication using [OAuth 2.0 providers](https://aura-stack-auth.vercel.app/docs/introduction) in an [Nuxt](https://nuxt.com/) application.

## Getting Started

You can run the documentation site locally for development or contribution.

### From the repository root

```bash
pnpm dev --filter=nuxt
```

### Or manually from the root directory

```bash
cd apps/nuxt
pnpm install
pnpm dev
```

Once started, open your browser at `http://localhost:3000` to view the app.
